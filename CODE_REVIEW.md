### 2. 体力系统 ⚠️

**审查结果：** 需优化

**发现问题：**

#### 问题 1: 体力恢复计算可能不准确
```javascript
// 当前代码 (第 67 行)
const recover = Math.floor((now - lastTime) / (30 * 60 * 1000))
```

**问题：** 如果用户长时间不玩 (如 1 天)，会一次性恢复大量体力，但上限是 5，计算浪费

**建议优化：**
```javascript
const recover = Math.floor((now - lastTime) / (30 * 60 * 1000))
const newEnergy = Math.min(this.data.maxEnergy, energy + recover)
// ✅ 已有上限保护，问题不大
```

**状态：** ✅ 已有保护，无需修改

#### 问题 2: 体力为 0 时游戏可能卡住
```javascript
// 当前代码 (第 75 行)
if (this.data.energy <= 0) {
  this.showAdForReward('energy')
  return false
}
```

**问题：** 如果用户不看广告，无法继续游戏

**建议优化：** 添加"等待恢复"选项

**状态：** ⚠️ 建议添加等待选项

## 🔧 建议修复

### 修复 M2: 广告失败备用方案

```javascript
// game.js 第 108 行修改
rewardedVideoAd.show().catch(() => {
  wx.showModal({
    title: '广告加载失败',
    content: '可以选择等待体力恢复 (30 分钟 +1)',
    confirmText: '查看体力',
    cancelText: '再试一次',
    success: (res) => {
      if (res.confirm) {
        wx.navigateBack({ delta: 1 })
      } else {
        this.watchAd(type)
      }
    }
  })
})
```

// game.js 第 75 行修改
