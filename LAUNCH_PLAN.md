## 🚀 上线步骤

### 第 1 步：注册小程序账号
```
1. 访问 https://mp.weixin.qq.com/
2. 点击"立即注册"
3. 选择"小程序"
4. 填写邮箱/密码
5. 邮箱验证
6. 填写主体信息 (个人/企业)
7. 填写管理员信息
8. 扫码绑定
9. 等待审核 (1-3 工作日)
```

### 第 2 步：完善小程序信息
```
1. 登录小程序后台
2. 设置 → 基本设置
3. 填写：
   - 名称：萌宠消消乐
   - 头像：512x512 PNG
   - 介绍：网红萌宠 + PK 对战的消除游戏
   - 服务类目：游戏 - 休闲
```

### 第 3 步：配置开发设置
```
1. 开发 → 开发管理
2. 获取 AppID
3. 下载开发者工具
4. 导入项目代码
5. 修改 app.json 中的 appid
6. 编译测试
```

### 第 4 步：接入流量主
```
条件：累计独立访客≥1000
步骤：
1. 流量主 → 开通流量主
2. 同意协议
3. 创建广告位
4. 代码中接入广告
5. 提交审核
```

### 第 5 步：提交审核
```
1. 版本 → 版本管理
2. 上传代码
3. 填写版本信息
4. 提交审核
5. 等待审核 (1-7 工作日)
6. 审核通过后发布
```
