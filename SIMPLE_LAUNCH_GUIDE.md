## 📋 上线步骤

### 第 1 步：注册小程序 ⏰ 30 分钟
```
https://mp.weixin.qq.com/
→ 立即注册 → 小程序 → 填写信息
```

### 第 2 步：下载开发者工具 ⏰ 20 分钟
```
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
→ 下载稳定版 → 安装
```

### 第 3 步：导入项目 ⏰ 10 分钟
```
目录：/home/admin/.openclaw/workspace/fruit-elimination-game/game-miniprogram
AppID：填入你的小程序 AppID
```

### 第 4 步：配置广告 ⏰ 30 分钟
```
条件：累计独立访客≥1000 (可先上线后补)

1. 小程序后台 → 流量主 → 开通
2. 创建广告位：激励视频广告
3. 复制广告 ID
4. 修改 game.js 第 97 行：
   adUnitId: '你的广告 ID'
```

### 第 5 步：测试编译 ⏰ 30 分钟
```
开发者工具 → 编译 → 测试 → 真机预览
```

### 第 6 步：提交审核 ⏰ 1-7 天
```
上传 → 提交审核 → 等待通过 → 发布
```
