## 📋 上线步骤

### 第 1 步：注册小程序账号 ⏰ 30 分钟

```
1. 访问 https://mp.weixin.qq.com/
2. 点击"立即注册"
3. 选择"小程序"
4. 填写邮箱 → 验证
5. 填写主体信息 (个人/企业)
6. 填写管理员信息
7. 扫码绑定
8. 等待审核 (1-3 工作日)
```

**所需材料：**
- 邮箱
- 身份证 (个人) / 营业执照 (企业)
- 手机号

---

### 第 2 步：完善小程序信息 ⏰ 20 分钟

```
1. 登录小程序后台
2. 设置 → 基本设置
3. 填写：
   - 名称：萌宠消消乐
   - 介绍：网红萌宠 + PK 对战的消除游戏，哈吉米/Doro/企鹅等 5 种主题
   - 头像：上传 512x512 PNG 图标
   - 服务类目：游戏 - 休闲
```

**图标建议：**
- 用哈吉米 (水豚) 形象
- 背景用渐变色
- 文字清晰

---

### 第 3 步：下载开发者工具 ⏰ 20 分钟

```
1. 访问 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 下载稳定版 (Stable Build)
3. 安装
4. 用小程序管理员微信扫码登录
```

---

### 第 4 步：导入项目 ⏰ 10 分钟

```
1. 打开微信开发者工具
2. 点击"+" 导入项目
3. 项目目录：
   /home/admin/.openclaw/workspace/fruit-elimination-game/game-miniprogram
4. AppID：填入你的小程序 AppID
5. 点击"导入"
```

---

### 第 5 步：配置广告 ⏰ 30 分钟

**条件：** 累计独立访客≥1000 (可先上线后补)

```
1. 登录小程序后台
2. 流量主 → 开通流量主
3. 同意协议
4. 创建广告位：
   - 激励视频广告：adunit-rewarded
   - 插屏广告：adunit-interstitial
5. 复制广告 ID
6. 修改 game.js 中的 adUnitId
```

**修改位置：**
```javascript
// game.js 第 215 行
adUnitId: 'adunit-rewarded' // 替换为你的广告 ID

// game.js 第 237 行
adUnitId: 'adunit-interstitial' // 替换为你的广告 ID
```

---

### 第 6 步：测试编译 ⏰ 30 分钟

```
1. 开发者工具 → 编译
2. 测试所有功能：
   - ✅ 消除玩法
   - ✅ 体力系统
   - ✅ 每日签到
   - ✅ 商店购买 (模拟)
   - ✅ 广告展示 (模拟)
   - ✅ PK 模式
   - ✅ 排行榜
3. 真机预览：
   - 点击"预览"
   - 扫码在手机上测试
```

---

### 第 7 步：提交审核 ⏰ 10 分钟

```
1. 开发者工具 → 上传
2. 填写版本号：1.0.0
3. 填写备注：萌宠消消乐 - 变现完整版
4. 登录小程序后台
5. 版本管理 → 开发版本
6. 选择刚上传的版本
7. 提交审核
8. 填写审核信息：
   - 功能介绍：消除类游戏
   - 测试账号：提供测试账号 (如需要)
```

**审核时间：** 1-7 工作日

---

### 第 8 步：发布上线 ⏰ 5 分钟

1. 审核通过后，小程序后台会收到通知
2. 版本管理 → 审核版本
3. 点击"发布"
