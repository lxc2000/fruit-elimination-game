## 🔧 已修复的问题

### 修复 1: 体力为 0 体验优化

**修复前：**
```javascript
if (this.data.energy <= 0) {
  this.showAdForReward('energy')
  return false
}
```

**修复后：**
```javascript
if (this.data.energy <= 0) {
  wx.showModal({
    title: '体力不足',
    content: '看视频广告 +3 体力，或等待自然恢复 (30 分钟 +1)',
    confirmText: '看广告',
    cancelText: '等待恢复',
    success: (res) => {
      if (res.confirm) {
        this.watchAd('energy')
      } else {
        wx.navigateBack({ delta: 1 })
      }
    }
  })
  return false
}
```

**效果：** 用户有选择权，体验更好
