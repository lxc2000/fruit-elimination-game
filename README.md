
# 水果消消乐 - 微信小游戏

一款简单有趣的休闲消除类微信小程序游戏。

## 游戏玩法
- 点击相邻的相同水果进行消除
- 消除的水果越多，获得的分数越高
- 每关有30步的限制，达到更高分数
- 提供提示和炸弹道具帮助通关

## 技术特点
- 纯原生微信小程序开发
- 响应式设计，适配各种屏幕
- 流畅的动画效果
- 本地存储最高分记录

## 项目结构
```
├── app.js              # 小程序入口文件
├── app.json            # 小程序配置文件
├── pages/
│   ├── index/          # 首页
│   ├── game/           # 游戏页面
│   └── result/         # 结果页面
├── utils/              # 工具函数
├── images/             # 图片资源
└── components/         # 组件
```

## 运行说明
1. 在微信开发者工具中导入项目
2. 配置小程序AppID
3. 点击编译即可运行

## 游戏截图
- 首页：渐变背景，开始游戏按钮，显示最高分
- 游戏页：8x8网格，分数和步数显示，道具按钮
- 结果页：显示得分，是否创新记录，操作按钮
